Limitations
  • The mass mailing system mlmmj accepts email messages where the email address of the mailing list is in the fields 'To' or 'Cc'. If the email address of the mailing list is the field 'Bcc' the system sends an email message with an explanation.
  • If you intend to make changes in the list of subscribers of a mailing list while a task for the mailing list is in progress, the MLMAdmin will give you such a warning:
    Warning: An active task is in progress, please wait.
    This helps avoid changes in the list of subscribers for running tasks.
